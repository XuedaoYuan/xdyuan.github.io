# CSS Grid 网格布局

[示例代码]( WebWorker Demo (opens new window))

# 一、概述

网格布局（Grid）是最强大的 CSS 布局方案。它将网页划分成一个个网格，可以任意组合不同的网格，做出各种各样的布局。比如说下面这种布局就适合用Grid。

Grid和Flex有一定的相似性，都可以指定容器里面的多个项的位置。

区别在于Flex是轴线布局，可以看成是一维布局。Grid布局可以把容器划分成 行 和 列， 然后有单元格的概念，可以理解为二维布局。Grid比Flex稍微复杂， 但是功能更加强大和丰富。

# 二、基本概念

# 2.1 容器和项目

<div class="container">
	<div class="item"><p></p></div>
	<div class="item"><p></p></div>
	<div class="item"><p></p></div>
</div>

采用网格布局的区域，叫做 容器（container）。容器内部采用网格定位的子元素， 叫做项目（item）。上面的div.container就是容器， div.item 就是项目。

项目只能是容器的顶层子元素，上述的p元素就不是项目。Grid布局只对项目起作用。

# 2.2 行和列

上面蓝色的水平区域的称为行，垂直区域的成为列。

# 2.3 单元格

行和列交叉的区域，称为单元格。 一般n行和m列会产生n*m个单元格。

# 2.4 网格线

划分网格的线， 称为网格线。水平网格线划分出行，垂直网格线划分出列。

一般n行有n + 1根水平网格线，m列有 m + 1根垂直网格线。

# 三、容器属性

Grid 布局属性分为两类， 一类作用在容器上， 称为容器属性， 一类作用在项目上面， 称为项目属性。

# 3.1 display属性

display: grid;指定一个容器采用网格布局。

div {
    display: grid;
}

上图是display: grid; 的效果。查看src/demo1.html。

默认情况下， 容器元素都是块级元素， 不过也可以改成行内元素

div {
    display: inline-grid;
}

指定div为行内元素， 但是元素内部还是采用网格布局。

上图是 display: inline-grid; 的效果,。查看 src/demo2.html

设置网格布局之后，容器子元素（项目）的float、display: inline-block、display: table-cell、vertical-align和column-* 等设置都会失效。

# 3.2 grid-template-columns、grid-template-rows

容器指定了网格布局以后，接着就要划分行和列。grid-template-columns属性定义每一列的列宽，grid-template-rows属性定义每一行的行高。

.container {
	display: inline-grid;
	grid-template-columns: 50px 50px 50px;
	grid-template-rows: 50px 50px 50px;
}

也可以使用百分比

.container {
	display: inline-grid;
	grid-template-columns: 33.33% 33.33% 33.33%;
	grid-template-rows: 33.33% 33.33% 33.33%;
}

# 3.2.1 repeat()

有时候，重复写同样的值非常麻烦，尤其网格很多时。这时，可以使用repeat()函数，简化重复的值。上面的代码用repeat()改写如下。

.container {
  display: grid;
  grid-template-columns: repeat(3, 33.33%);
  grid-template-rows: repeat(3, 33.33%);
}

repeat()接受两个参数，第一个参数是重复的次数（上例是3），第二个参数是所要重复的值。

repeat()重复某种模式也是可以的。

grid-template-columns: repeat(2, 100px 20px 80px);

效果如上图。查看 src/demo3.html

# 3.1.2 auto-fill 关键字

有时候， 单元格的大小是固定的，但是容器的大小是不一定的。 如果希望某一行容纳尽可能多的单元格， 可以使用auto-fill 关键字表示自动填充。

.container {
	display: grid;
	grid-template-columns: repeat(auto-fill, 100px);
	width: 550px;
	border: 4px dashed #ccc;
}

查看 src/demo4.html

# 3.1.3 fr 关键字

fr关键字用来表示比例关系。fraction 片段的意思。 有点类似于flex属性。比如1fr和2fr， 表示后者是前者的2倍.

.container {
	display: grid;
	width: 200px;
	grid-template-columns: 1fr 1fr;
	border: 4px dashed #ccc;
}

表示两个相同宽度的列。 查看src/demo5.html。

fr 也可以和绝对长度的单位结合使用，这个比较实用。

查看 src/demo6.html。

# 3.1.4 minmax()

minmax()函数产生一个长度范围，表示长度就在这个范围之中。它接受两个参数，分别为最小值和最大值。

.container {
	display: grid;
	width: 200px;
	grid-template-columns: 1fr 1fr minmax(100px, 1fr);
	border: 4px dashed #ccc;
}

查看 src/demo7.html

# 3.1.5 auto 关键字

表示由浏览器自己决定长度， 比如

grid-template-columns: 100px auto 100px;

表示第一列和第三列的宽度是100px， 第二列的宽度自定义。 基本上就等于剩下的最大宽度。除非单元格内容设置了min-width， 且这个值大于最大宽度。

查看 src/demo8.html.

# 

grid-template-columns属性和grid-template-rows属性里面，还可以使用方括号，指定每一根网格线的名字，方便以后的引用。

.container {
  display: grid;
  grid-template-columns: [c1] 100px [c2] 100px [c3] auto [c4];
  grid-template-rows: [r1] 100px [r2] 100px [r3] auto [r4];
}

上述代码表示3行3列， 所以分别有4根垂直和水平网格线，方括号里的就是八根线的名字。

同一根网格线允许有多个名字， 比如 [r5 the-fifth]

# 3.1.7 布局实例

# （1）两列布局
.container {
	display: grid;
	grid-template-columns: 200px 1fr;
    grid-template-columns: 200px auto;
	/* grid-template-columns: 70% 30%; */
	height: 100%;
}

查看src/demo9.html

# （2）传统十二网格布局
grid-template-columns: repeat(12, 1fr);

查看src/demo10.html

# 3.2 row-gap、column-gap、gap 属性

row-gap属性设置行与行的间隔（行间距），column-gap属性设置列与列的间隔（列间距）。

.container {
	display: grid;
	grid-template-columns: 50px 50px 50px;
	grid-template-rows: 50px 50px 50px;
	column-gap: 20px;
	row-gap: 20px;
}

查看 src/demo11.html

以前叫 grid-row-gap、grid-column-gap、grid-gap， 现在去除了grid-。

gap相当于是row-gap、column-gap的简写。

gap: <row-gap> <column-gap>;

如果gap省略了第二个值， 默认第二个值等于第一个值。

# 3.3 grid-template-areas 属性

网格布局允许指定区域（area），一个区域可以由一个或者多个单元格组成。 grid-template-areas用于定于区域。

.container {
	display: grid;
	grid-template-columns: 50px 50px 50px;
	grid-template-rows: 50px 50px 50px;
	grid-template-areas: "a b c" "d e f" "g h i";
}

上述代码先分出九个单元格， 然后将其命名问a到i的九个区域， 分别对应着九个单元格。多个单元格合并成一个的写法如下

grid-template-areas: 'a a a' 'b b b' 'c c c';

上面代码将9个单元格分成a、b、c三个区域。

下面是一个实例，查看 src/demo12.html

grid-template-areas: "header header header"
                     "main main sidebar"
                     "footer footer footer";

上述代码中， 顶部是header， 中间是main和sidebar，底部是footer。

如果某些区域不需要利用，则使用"点"（.）表示。

grid-template-areas: 'a . c'
                     'd . f'
                     'g . i';

上面代码中，中间一列为点，表示没有用到该单元格，或者该单元格不属于任何区域。

注意，区域的命名会影响到网格线。每个区域的起始网格线，会自动命名为区域名-start，终止网格线自动命名为区域名-end。比如，区域名为header，则起始位置的水平网格线和垂直网格线叫做header-start，终止位置的水平网格线和垂直网格线叫做header-end。

# 3.4 grid-auto-flow 属性

划分网格以后， 容器的子元素会按照顺序， 顺序的放在每一个网格。 默认先行后列。grid-auto-flow的默认值是row。也可以设置成column。如下图

现在修改为grid-auto-flow: row dense， 这个值表示先行后列， 同时尽可能紧密填满，不出现空格。

# 3.5 justify-items 属性，align-items 属性，place-items 属性

justify-items属性设置单元格内容的水平位置（左中右），align-items属性设置单元格内容的垂直位置（上中下）。

.container {
  justify-items: start | end | center | stretch;
  align-items: start | end | center | stretch;
}

这两个属性的写法完全相同，都可以取下面这些值。

  • start：对齐单元格的起始边缘。

  • end：对齐单元格的结束边缘。

  • center：单元格内部居中。

  • stretch：拉伸，占满单元格的整个宽度（默认值）。

    查看 src/demo15.html

place-items属性是align-items属性和justify-items属性的合并简写形式。如果省略第二个值，则浏览器认为与第一个值相等。

place-items: <align-items> <justify-items>;

# 3.6 justify-content 属性，align-content 属性，place-content 属性

justify-content属性是整个内容区域在容器里面的水平位置（左中右），align-content属性是整个内容区域的垂直位置（上中下）。

.container {
  justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
  align-content: start | end | center | stretch | space-around | space-between | space-evenly;  
}

直接查看 src/demo16.html

  • stretch - 项目大小没有指定时，拉伸占据整个网格容器。
  • space-around - 每个项目两侧的间隔相等。所以，项目之间的间隔比项目与容器边框的间隔大一倍。
  • space-between - 项目与项目的间隔相等，项目与容器边框之间没有间隔。
  • space-evenly - 项目与项目的间隔相等，项目与容器边框之间也是同样长度的间隔。

place-content属性是align-content属性和justify-content属性的合并简写形式。

place-content: <align-content> <justify-content>

# 3.7 grid-auto-columns 属性，grid-auto-rows 属性

有时候，一些项目的指定位置，在现有网格的外部。比如网格只有3列，但是某一个项目指定在第5行。这时，浏览器会自动生成多余的网格，以便放置项目。

grid-auto-columns属性和grid-auto-rows属性用来设置，浏览器自动创建的多余网格的列宽和行高。

查看 src/demo17.html

# 3.8 grid-template 属性，grid 属性

grid-template属性是grid-template-columns、grid-template-rows和grid-template-areas这三个属性的合并简写形式。

grid属性是grid-template-rows、grid-template-columns、grid-template-areas、 grid-auto-rows、grid-auto-columns、grid-auto-flow这六个属性的合并简写形式。

从易读易写的角度考虑，还是建议不要合并属性，所以这里就不详细介绍这两个属性了。

# 四、项目属性

# 4.1 grid-column-start 属性，grid-column-end 属性，grid-row-start 属性，grid-row-end 属性

项目的位置是可以指定的，具体方法就是指定项目的四个边框，分别定位在哪根网格线。

  • grid-column-start属性：左边框所在的垂直网格线
  • grid-column-end属性：右边框所在的垂直网格线
  • grid-row-start属性：上边框所在的水平网格线
  • grid-row-end属性：下边框所在的水平网格线

查看 src/demo18.html

.item:nth-child(1) {
	background-color: red;
	grid-column-start: 2;
	grid-column-end: 4;
}

上述代码表示， 1号项目的左边框是第二根垂直网格线，右边框是第四根垂直网格线。

再看一个效果

.item:nth-child(1) {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 2;
  grid-row-end: 4;
}

这四个属性的值，除了指定为第几个网格线，还可以指定为网格线的名字。就是我们之前说的可以给网格线取名字。

这四个属性的值还可以使用span关键字，表示"跨越"，即左右边框（上下边框）之间跨越多少个网格。

.item:nth-child(1) {
  grid-column-start: span 2;
}

查看 src/demo18.html

# 4.2 grid-column 属性，grid-row 属性

grid-column属性是grid-column-start和grid-column-end的合并简写形式，grid-row属性是grid-row-start属性和grid-row-end的合并简写形式。 不再阐述了。看一下下面的例子

.item-1 {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}
/* 等同于 */
.item-1 {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 1;
  grid-row-end: 2;
}

# 4.3 grid-area 属性

grid-area属性指定项目放在哪一个区域。

查看 src/demo19.html

.container {
	display: grid;
	grid-template-columns: repeat(3, 100px);
	grid-template-rows: repeat(3, 100px);
	border: 4px dashed #ccc;
	/* 划分区域 */
	grid-template-areas: 'a b c' 'd e f' 'g h i';
}
.item:nth-child(1) {
	background-color: red;
	grid-area: e;
}

grid-area属性还可用作grid-row-start、grid-column-start、grid-row-end、grid-column-end的合并简写形式，直接指定项目的位置。

.item {
  grid-area: <row-start> / <column-start> / <row-end> / <column-end>;
}

比如

.item-1 {
  grid-area: 1 / 1 / 3 / 3;
}

# 4.4 justify-self 属性，align-self 属性，place-self 属性

justify-self属性设置单元格内容的水平位置（左中右），跟justify-items属性的用法完全一致，但只作用于单个项目。

align-self属性设置单元格内容的垂直位置（上中下），跟align-items属性的用法完全一致，也是只作用于单个项目。

.item {
  justify-self: start | end | center | stretch;
  align-self: start | end | center | stretch;
}
  • start：对齐单元格的起始边缘。
  • end：对齐单元格的结束边缘。
  • center：单元格内部居中。
  • stretch：拉伸，占满单元格的整个宽度（默认值）。

place-self属性是align-self属性和justify-self属性的合并简写形式。

place-self: <align-self> <justify-self>;

如果省略第二个值，place-self属性会认为这两个值相等。
